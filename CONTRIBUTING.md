# Contributing

Thanks for considering a contribution!

1. Fork the repo and create your branch from `main`.
2. Run `npm install` inside `app/` and make sure `npm test` and `npm run lint` pass.
3. Open a pull request describing the change and why it's needed.
4. All PRs run through the CI pipeline defined in `.github/workflows/ci.yml` — it must pass before merge.

Bug reports and feature requests are welcome via GitHub Issues.
